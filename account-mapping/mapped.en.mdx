## The Mapped Panel

The right panel displays all successfully mapped accounts organized in a hierarchical taxonomy tree. Each taxonomy category can be expanded or collapsed using the disclosure triangle.

## Table Columns

| Column | Description |
|---|---|
| **Checkbox** | Select accounts for bulk actions |
| **Position Name** | Taxonomy position in a hierarchical tree |
| **Current Year** | Currency value (editable) |
| **Previous Year** | Currency value (editable) |
| **Actions** | Three-dot menu with account actions |

## Filter by Category

Use the category filter buttons at the top of the mapped panel to show only accounts within a specific taxonomy category. The **"All"** button resets the filter.

## Account Actions

Click the three-dot menu (⋯) on any mapped account row:

- **Remove Mapping** — Unmap the account, returning it to the "Open" panel
- **Edit** — Open the account editor to change name, number, or values
- **Hybrid Account** — Create separate mappings for current and previous year
- **Archive** — Archive the account

## Category Actions

Click the three-dot menu (⋯) on a taxonomy category row:

- **Expand All Child Categories** — Expand the entire subtree
- **Collapse All Child Categories** — Collapse the entire subtree

## Bulk Actions

From the actions menu at the top:

- **Edit Target Values** — Open the target values editor
- **Remove Mapping from Selected** — Unmap all checked accounts
- **Expand all / Collapse all** — Toggle the entire tree
- **Select all / Unselect all** — Toggle all checkboxes

## Summary

Below the table, a summary section shows aggregate totals for each top-level taxonomy category.

## Export

Click the **download icon** in the mapped panel header to export all mapped accounts with their taxonomy positions and values.
