## Konten filtern

Klicken Sie auf den **„Filter"**-Button, um das Filterpanel zu öffnen. Filter gelten sowohl für offene als auch zugeordnete Konten.

## Verfügbare Filterkriterien

| Kriterium | Operationen |
|---|---|
| **Kontoname** | Enthält, Gleich, Beginnt mit |
| **Kontonummer** | Enthält, Gleich, Beginnt mit |
| **Wert aktuelles Jahr** | =, >, <, ≥, ≤ |
| **Wert Vorjahr** | =, >, <, ≥, ≤ |

## Filter verwenden

<Steps>
  <Step title="Regel hinzufügen">
    Klicken Sie auf **„Neue Regel hinzufügen"**, um eine Filterbedingung zu erstellen.
  </Step>
  <Step title="Regel konfigurieren">
    Wählen Sie das Feld, die Operation und geben Sie den zu suchenden Wert ein.
  </Step>
  <Step title="Anwenden">
    Klicken Sie auf **„Filter anwenden"**. Die geschätzte Trefferanzahl wird während der Berechnung angezeigt.
  </Step>
</Steps>

## Filter kombinieren

Mehrere Filterregeln können kombiniert werden. Alle Regeln müssen zutreffen, damit ein Konto angezeigt wird (UND-Logik).

## Filter zurücksetzen

- **Zurücksetzen** — Einzelne Regel löschen
- **Alle Regeln zurücksetzen** — Alle Filterregeln auf einmal löschen
