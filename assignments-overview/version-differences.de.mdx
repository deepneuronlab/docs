## Neue Versionen eines Dokuments

Eine neue Version eines **Prüfungsdokuments** oder **Nachweisdokuments** kann über den **Dateimanager** hochgeladen werden.

Durch Klicken auf **„Dokumente hochladen"** in der oberen rechten Ecke des Dateimanagers kann ausgewählt werden, ob eine **neue Version** eines **Prüfungsdokuments** oder eines **Nachweisdokuments** hinzugefügt werden soll. Neben allen aufgelisteten Dokumenten befindet sich zudem die Schaltfläche **„Neue Version hochladen"**, über die ein Dokument direkt vom Computer hinzugefügt werden kann.

Die App hebt automatisch die Unterschiede zwischen den Dokumentversionen hervor. Über den Tab **„Versionsunterschiede anzeigen"** oberhalb der Dokumentansicht gelangen Sie in den Vergleichsmodus, in dem beide Versionen nebeneinander angezeigt werden.

### Vergleichsmodus

Im Vergleichsmodus werden beide Dokumentversionen nebeneinander mit synchronisiertem Scrollen angezeigt. Wenn mehrere Dokumentpaare verfügbar sind, wählen Sie zunächst aus, welches Paar verglichen werden soll.

Unterschiede werden in drei Kategorien eingeteilt:

- **Redaktionell** — Kleinere Formulierungs- oder Formatierungsänderungen
- **Inhaltlich (erwartet)** — Wesentliche Änderungen, die erwartet wurden
- **Inhaltlich (unerwartet)** — Wesentliche Änderungen, die einer genaueren Überprüfung bedürfen

Über die Filtersteuerung in der Vergleichssymbolleiste können die angezeigten Annotationen nach Klassifikation gefiltert werden. Annotationen werden in beiden Dokumentansichten hervorgehoben. Durch Klicken auf eine Annotation wird die gegenüberliegende Ansicht zur entsprechenden Änderung gescrollt.

<Note>
Der Versionsvergleich kann einen Moment dauern. Während der Verarbeitung wird eine Ladeanzeige eingeblendet. Der Verarbeitungsstatus durchläuft die Stufen `not_started`, `in_progress` und `ready`. Falls die Verarbeitung fehlschlägt, wird der Status auf `failed` gesetzt und Sie können es erneut versuchen.
</Note>

Diese Funktion stellt sicher, dass alle Änderungen zwischen den Dokumentversionen klar ersichtlich sind und überprüft werden können.

Nach dem Hochladen einer neuen Dokumentversion kann es vorkommen, dass einige **Referenzen** nicht mehr gefunden werden. In diesem Fall erhält die Frage in der **Prüfungsübersicht** den Status **„erneut prüfen"**, gekennzeichnet durch eine **lila** Markierung. Wird die Referenz im Dokument erkannt, wird sie **blau** markiert.

Das Hochladen einer neuen Dokumentversion wirkt sich direkt auf den Fortschritt in der **Checkliste** aus. Nach dem Hochladen einer neuen Version eines Prüfungsdokuments werden alle beantworteten Fragen auf **KI-Vorschlag** zurückgesetzt.

<Warning>
**Hinweis:** Das Hochladen der neuen Version muss durch Klicken auf **„Bestätigen"** in der unteren rechten Ecke des Fensters bestätigt werden. Vor der Bestätigung ist es noch möglich, die hochgeladene Version zu löschen. Nach der Bestätigung kann die neu hinzugefügte Version nicht mehr gelöscht werden, und der Prüfung wird automatisch aktualisiert.
</Warning>
