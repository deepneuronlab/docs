## Export Formats

DNL Audit supports export in three formats:

<CardGroup cols={3}>
  <Card title="PDF" icon="file-pdf">
    Audit documents with annotations and markings
  </Card>
  <Card title="Word" icon="file-word">
    Editable documents with comments and proposed changes
  </Card>
  <Card title="Excel" icon="file-excel">
    Tabular overviews of the numbers audit
  </Card>
</CardGroup>

## Performing an Export

<Steps>
  <Step title="Open Assignment">
    Navigate to the desired assignment in the **Assignments Overview**.
  </Step>
  <Step title="Start Export">
    Open the **Export menu** and select the desired format.
  </Step>
  <Step title="Configure Options">
    Depending on the export format, you can set additional options (e.g., which content to include).
  </Step>
  <Step title="Download">
    The export is generated. Once ready, the file is automatically available for download.
  </Step>
</Steps>

## Export Contents

### PDF Export
In the exported PDF, all figures from the numbers section are marked: **green** (confirmed), **red** (rejected), or **blue** (unchecked). Clicking a mark opens a comment with information about the audit results.

### Excel Export
The Excel export contains a tabular overview of all audited figures with their results, evidence, and remarks.

### Word Export
The Word export contains the audit document with all comments and proposed changes created and saved in Edit Mode.
