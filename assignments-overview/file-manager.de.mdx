## 1. Verwalten von Dateien im Dateimanager

Der Dateimanager kann in der Auftragsübersicht über das Drei-Punkte-Menü (⋯) geöffnet werden, das sich neben „Vorbereiten" und „Reviewen" befindet. Alternativ lässt sich der Dateimanager innerhalb von ai/checklist und ai/numbers unter dem Tab Dokument aufrufen.

Der Dateimanager ermöglicht es, prüfungsrelevante Dokumente wie den Anhang, Lagebericht, oder Nachweisdokumente hochzuladen, zu sortieren, herunterzuladen, oder zu archivieren, beziehungsweise zu löschen.

Hochgeladene Dokumente werden automatisch nach dem vom Prüfer angegebenen Dokumenttyp strukturiert. Alle Dokumente sind unter den Tabs "Prüfungsdokumente" oder "Nachweisdokumente" zu finden.

### Unterstützte Dateiformate

Folgende Dateiformate werden für den Upload akzeptiert:

| Format | Endungen |
|---|---|
| PDF | `.pdf` |
| Word (OOXML) | `.docx` |

<Warning>
Das veraltete `.doc`-Format (Microsoft Word 97–2003) wird **nicht unterstützt**. Bitte konvertieren Sie `.doc`-Dateien vor dem Hochladen in das `.docx`-Format.
</Warning>

<Note>
Excel-Dateien (`.xlsx`, `.xls`, `.xlsm`) können als Nachweisdokumente akzeptiert werden, wenn die Excel-Unterstützung für Ihre Organisation aktiviert ist.
</Note>

## 2. Aktionen im Dateimanager

Jedes Dokument kann:

- im Vorschaubereich angezeigt werden
- bearbeitet werden (Dokumenttyp und Dateiname)
- heruntergeladen werden
- archiviert und anschließend gelöscht werden

Im Bereich der Nachweisdokumente ist es außerdem möglich, [Referenzen aus anderen Aufträgen zu importieren](/checklist/references.de#referenzen-aus-einem-anderen-auftrag-importieren). Es werden nur Referenzen hinzugefügt, die im Zieldokument gefunden werden können.

<Warning>
**Hinweis:** Prüfungsdokumente können durch das Hochladen einer neuen Dokumentenversion ersetzt werden. Das Hochladen einer neuen Version wirkt sich direkt auf den Fortschritt der Zahlenprüfung aus. Änderungen an Zahlen im Vergleich zur vorherigen Version werden erkannt, und die entsprechenden Zahlen werden wieder auf ungeprüft gesetzt. Der Prüfer muss diese Zahlen erneut bestätigen.

Sobald eine neue Version hochgeladen wurde, ist es nicht mehr möglich, zu einer früheren Version zurückzukehren. Dies gewährleistet die Vollständigkeit und Genauigkeit im Prüfungsprozess.
</Warning>

## 3. Archivierte Dokumente

Im Bereich "Archivierte Dokumente" können Prüfer:

- archivierte Dokumente anzeigen,
- sie wiederherstellen,
- oder dauerhaft löschen.

## 4. Löschen von Dokumenten

Ein Dokument kann nur gelöscht werden, nachdem es archiviert wurde. Diese Sicherheitsmaßnahme verhindert das versehentliche Löschen wichtiger Dateien, oder den möglichen Verlust des Prüfungsfortschritts.
