## 1. Expand All

The **"Expand All" button** in the action bar above the Checklist section opens all questions simultaneously. Next to it, the **"Collapse All" button** closes all questions again.

## 2. Document View Mode

Users can choose between the **"Dual View"** or the **"Single View" Mode** in the action bar above the Checklist section. Switching between the two modes allows working with two documents side by side, facilitating comparisons and quick actions. For more focus, single view mode can be chosen.

## 3. Single Question Mode

By switching to **single question mode,** only the currently selected question will be displayed.

## 4. Filter Questions

The **"filter" button** is located in the top-left area of the App, above the Checklist section. Questions can be filtered based on different criteria, such as **status**, **answers** and **references**. Questions can also be filtered through **combined criteria**.

## 5. Flag Questions

Questions can be flagged for re-evaluation by clicking the **flag button** on the bottom left corner within the question. The question will then be highlighted in **purple**.

Flags can be removed by clicking the flag button again. All flags can be cleared at once using the **"Unflag" button** in the action bar above the checklist section.

<Note>
**Note:** Flagged questions cannot be reviewed. To set the review-checkmark, the question must be unflagged and checked.
</Note>

## 6. Progress Bar

The **progress bar** next to the **"filter" button** provides the total number of questions and displays how many questions fall into each of the defined categories. The **progress bar** shows the overall progress and indicates how many questions are:

- **prepared**
- **flagged by the user**
- **flagged by Notes Auditor**
- **reviewed**
- **not started**
