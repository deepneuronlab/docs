In **review mode**, all features available in **prepare mode** remain fully accessible. A reviewer has permission to check and review all questions in ai/checklist.

Questions can be marked as reviewed using the **double-check icon** (✓✓), supporting a systematic completion of the review process.

All questions of one category can be reviewed simultaneously by selecting the **"Review All" checkboxes.** This action can be reversed.

<Warning>
**Note:** Once a question is marked as reviewed, it is no longer editable in this status. Any changes require the removal of the review mark before continuing with the audit.
</Warning>
