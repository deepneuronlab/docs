### FileDocumentDiff
| Feld | Typ | Beschreibung |
|---|---|---|
| id | string | Eindeutige Diff-ID |
| fileBeforeId | string | ID des älteren Dokuments |
| fileBeforeName | string \| null | Name des älteren Dokuments |
| fileAfterId | string | ID des neueren Dokuments |
| fileAfterName | string \| null | Name des neueren Dokuments |
| state | enum | Verarbeitungsstatus: `not_started`, `in_progress`, `ready`, `failed` |
| createdAt | datetime | Erstellungszeitpunkt |
| updatedAt | datetime | Letzte Aktualisierung |

### DocumentDiff (Versionsvergleich)
| Feld | Typ | Beschreibung |
|---|---|---|
| new_document_id | string | ID des neueren Dokuments |
| old_document_id | string | ID des älteren Dokuments |
| state | enum | Verarbeitungsstatus: `not_started`, `in_progress`, `ready`, `failed` |
| new_annotations | Map | Annotationen im neueren Dokument |
| old_annotations | Map | Annotationen im älteren Dokument |

Jede Diff-Annotation enthält:
- **kind**: `added`, `deleted` oder `unchanged`
- **classification**: `editorial`, `substantive_expected` oder `substantive_unexpected`
- **annotation**: Position (pageIndex, boundingBox, rects), Text und verknüpfte IDs
