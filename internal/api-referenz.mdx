Alle Endpunkte nutzen das Prefix `/anexo`. Authentifizierung erfolgt via Bearer Token (OAuth2).

## Authentifizierung & Benutzer

| Methode | Pfad | Beschreibung |
|---|---|---|
| POST | `/swagger-auth/login` | Login |
| POST | `/users/register` | Registrierung |
| POST | `/users/reset-password` | Passwort zurücksetzen |
| GET | `/users` | Benutzer auflisten |
| PUT | `/users/{id}` | Benutzer aktualisieren |
| PUT | `/users/locale` | Sprache ändern (de-DE, en-GB, sv-SE, cs-CZ, es-ES, fr-FR, it-IT, pt-PT) |

## OIDC / SSO

|---|---|---|

| GET | `/mazars/login-uri` | Mazars Login-Redirect |

## Admin-Panel

| Methode | Pfad | Beschreibung |
|---|---|---|
| GET | `/admin/groups/list` | Gruppen auflisten |
| POST | `/admin/groups/create` | Gruppe erstellen |
| POST | `/admin/users/create-bulk` | Benutzer massenhaft erstellen |
| POST | `/admin/users/verify` | Benutzer verifizieren |
| PATCH | `/admin/users/{id}/toggle-active` | Benutzer aktivieren/deaktivieren |
| PATCH | `/admin/users/{id}/toggle-group-admin` | Gruppenadmin umschalten |
| GET | `/admin/checklists/list` | Checklisten verwalten |
| POST | `/admin/checklists/toggle` | Checkliste aktivieren/deaktivieren |
| GET | `/admin/features/group/{id}` | Feature Flags pro Gruppe |
| POST | `/admin/features/toggle` | Feature Flag umschalten |

## Mandate (Companies)

|---|---|---|

| POST | `/companies` | Mandat erstellen |

| DELETE | `/companies/{id}` | Mandat löschen |

## Aufträge (Assignments)

| Methode | Pfad | Beschreibung |
|---|---|---|
| GET | `/companies/{cid}/assignments` | Aufträge auflisten |
| POST | `/companies/{cid}/assignments` | Auftrag erstellen |
| PUT | `/companies/{cid}/assignments/{id}` | Auftrag bearbeiten |
| PATCH | `/companies/{cid}/assignments/{id}/status` | Status ändern |
| PATCH | `/companies/{cid}/assignments/{id}/users` | Benutzer zuweisen |
| GET | `/companies/{cid}/assignments/{id}/details` | Details abrufen |

## Checkliste (Questions)

| Methode | Pfad | Beschreibung |
|---|---|---|
| GET | `/assignments/{aid}/questions` | Fragen auflisten |
| GET | `/assignments/{aid}/questions/{id}` | Frage-Details |
| PATCH | `/assignments/{aid}/questions/{id}` | Antwort aktualisieren |

**Flow-Status:** `UNEDITED` → `AUDITING` → `AUDITED` → `REVIEWING` → `REVIEWED`
