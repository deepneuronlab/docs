## AI-generated References

AI generated answers and suggestions **refer to content** from uploaded documents. These references are listed in the question section, and are highlighted directly in the document.

To view the relevant text passage, click **"View"** next to the reference. The system will automatically open the page and highlight the reference in **blue** (AI-answer) or **yellow** (AI-suggestion).

It is also possible to view the linked question for any highlighted reference in the document by **right-clicking** on the **reference** to see question details. By clicking **"View"** the question in the checklist will open automatically.

## Add References

References can be added by selecting the desired text in the document and choosing **"Add as reference"** from the appearing menu box. If a text passage has already been used as a reference for another question, it can still be **added to another question.**

<Note>
**Note:** Manually adding references from the document is only possible if the respective question is opened in the Checklist section. If no question is open, the system won't be able to know which question the reference should be connected to, making addition of references impossible.
</Note>

## Delete References

To **delete** a reference, click on the **bin** icon next to the reference within the references box in the Checklist section.
