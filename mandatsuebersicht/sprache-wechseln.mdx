## Change the Application Language

DNL Audit supports multiple languages. The language setting affects the entire user interface, including menus, buttons, and system messages.

### Supported languages

| Language | Locale code |
|---|---|
| Deutsch (German) | `de-DE` |
| English | `en-GB` |
| Français (French) | `fr-FR` |
| Italiano (Italian) | `it-IT` |
| Español (Spanish) | `es-ES` |
| Português (Portuguese) | `pt-PT` |

<Steps>
  <Step title="Open Menu">
    Click on your **profile menu** in the top right corner of the application.
  </Step>
  <Step title="Select Language">
    Choose the desired language from the available list.
  </Step>
  <Step title="Apply Change">
    The application will immediately display in the selected language. The setting is saved to your user account.
  </Step>
</Steps>

<Note>
The language setting is stored both in your user profile and locally in the browser. Your preferred language will be used automatically on next login.
</Note>
