## Assigning Users to an Assignment

Each assignment can be assigned to multiple users. Assignment is done through the assignment settings.

<Steps>
  <Step title="Open Assignment Settings">
    Navigate to the desired assignment in the **Assignments Overview** and open **Access Rights**.
  </Step>
  <Step title="Add or Remove Users">
    Select the users who should have access to the assignment. You can also remove users.
  </Step>
</Steps>

## Permissions

Available permissions depend on your role within the organization:

| Permission | Description |
|---|---|
| **Create Clients** | Create new clients |
| **Edit Clients** | Edit existing clients |
| **Delete Clients** | Remove clients and associated assignments |
| **Create Assignments** | Create new assignments within a client |
| **Delete Assignments** | Remove existing assignments |
| **Manage Users** | Change user assignments for assignments |
| **View All Clients** | Access all clients in the organization |
| **View All Assignments** | Access all assignments in the organization |

<Note>
Group administrators can manage users and their permissions via the Admin Panel. Contact your group administrator if you need additional permissions.
</Note>
