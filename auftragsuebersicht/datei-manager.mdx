## 1. Managing Files in the File Manager

The File Manager can be accessed in the top right corner located next to the **"Help / Feedback" Button**.

The File Manager allows users to upload, sort, download, and archive or delete audit-relevant documents such as Notes, Management Reports, and Supporting Documents.

Uploaded documents are automatically organized according to the document type specified by the user. All documents can be found under the tabs "Audit Documents" and "Supporting Documents".

## 2. Actions in the File Manager

Each document can be:

- viewed in the preview section
- edited (document type and name can be changed)
- downloaded
- archived, and subsequently deleted

For Supporting Documents, it is also possible to import references from other assignments.

<Warning>
**Note:** Audit Documents can be replaced by uploading a new version. Uploading a new version directly affects the progress in ai/numbers. Any changes in figures compared to the previous version will be detected, and the corresponding figures will be reset to *unchecked*. The auditor must re-confirm these figures.

Once a new version has been uploaded, it is not possible to revert to a previous version. This ensures completeness and accuracy within the audit process.
</Warning>

## 3. Archived Documents

Within the Archived Documents section, users can:

- view archived documents
- restore them, or
- permanently delete them.

## 4. Deleting Documents

A document can only be deleted after it has been archived. This security measure prevents accidental deletion of important files and the potential loss of audit progress.
