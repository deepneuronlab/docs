## Working with Word Files and PDF Files

Word files and PDF files are handled differently within the App, due to the distinct characteristics of each file format. The way **comments** and **proposed changes** are displayed, edited, and exported depends on the type of document used for the audit.

To switch from the default **Audit Mode** to the **Edit Mode**, the **"Edit" button** on the right corner, above the document view, must be clicked. To exit the Edit Mode simply click on **"Exit Edit"**.

## 1. Working with Word Files

- Word files, including all existing comments and proposed changes can be fully imported/uploaded.
- Existing and newly added comments and proposed changes are displayed on the right side in the document section.
- New comments are created by selecting text, opening the comment field, entering the comment, and posting it via the **red arrow icon**. A final save action is required to store the comment permanently.
- Proposed changes are entered directly into the text and saved with the **"save" icon** in the top-right corner.
- Separate buttons above the editing area allow switching between comments and proposed changes.
- Both comments and proposed changes will be part of the exported document once they are saved.

## 2. Working with PDF Files

- When working with PDF files, any comments or proposed changes previously added in the original file will **not** be imported.
- Adding new comments and proposed changes to the document is possible, by highlighting the preferred section within the document.
- A menu box allows a choice of adding a comment or a proposed change. By clicking on either option, a text box opens, and the comment or change can be entered and saved by clicking the **"save"** button.
- Unlike working with Word files in the Edit Mode, the comments and changes will not be displayed beside the document, but the selected text will be highlighted **yellow** (comment) or underlined in **blue** (proposed changes).
- Both comments and proposed changes will be part of the exported document once they are saved.
