## Filter Accounts

Click the **"Filter"** button to open the filter panel. Filters apply to both the unmapped and mapped accounts panels.

## Available Filter Criteria

| Criterion | Operations |
|---|---|
| **Account Name** | Contains, Equals, Starts with |
| **Account Number** | Contains, Equals, Starts with |
| **Current Year Value** | =, >, <, ≥, ≤ |
| **Previous Year Value** | =, >, <, ≥, ≤ |

## Using Filters

<Steps>
  <Step title="Add a rule">
    Click **"Add a new rule"** to create a filter condition.
  </Step>
  <Step title="Configure the rule">
    Select the field, operation, and enter the value to match.
  </Step>
  <Step title="Apply">
    Click **"Apply Filters"**. The estimated match count is shown while the filter is being calculated.
  </Step>
</Steps>

## Combine Filters

Multiple filter rules can be combined. All rules must match for an account to be shown (AND logic).

## Reset Filters

- **Reset** — Clears a single rule
- **Reset all rules** — Clears all filter rules at once
