## Neue Version eines Dokuments hochladen

Eine neue Version eines Prüfungsdokuments oder eines Nachweisdokuments kann über den **Dateimanager** zur Prüfung hinzugefügt werden.

Durch einen Klick auf **"Dokumente hochladen"** in der oberen rechten Ecke im Dateimanager kann der Nutzer **eine neue Version eines Prüfungsdokuments oder eines Nachweisdokuments** hinzufügen. Die Schaltfläche **"Neue Version hochladen"** befindet sich neben allen aufgelisteten Dokumenten. Über diese Schaltfläche kann eine Datei direkt vom Computer hinzugefügt werden.

Das Hochladen neuer Dokumentversionen kann dazu führen, dass Zahlen im **Zahlenbereich** verloren gehen. In diesem Fall wird die entsprechende Zahl durchgestrichen und die App bietet die Option **"Neu verbinden"** an. Durch einen Klick auf **"Neu verbinden"** kann die aktualisierte Zahl im Dokument markiert und wieder mit der Prüfung verknüpft werden.

Unterschiede zwischen den Dokumentversionen werden automatisch hervorgehoben. Über den Reiter **"Versionsunterschiede"** oberhalb der Dokumentansicht gelangen Sie in den Vergleichsmodus, in dem Änderungen nebeneinander überprüft werden können.

Beim Vergleich werden Unterschiede als **redaktionell**, **inhaltlich (erwartet)** oder **inhaltlich (unerwartet)** klassifiziert. Sie können die Annotationen nach Klassifikation filtern, um sich auf die relevantesten Änderungen zu konzentrieren. Ein Klick auf den markierten Text oder die markierten Zahlen öffnet die entsprechende Zahl im Zahlenbereich.

<Note>
Der Versionsvergleich kann einen Moment dauern. Während der Verarbeitung wird eine Ladeanzeige eingeblendet.
</Note>

Diese Funktion stellt sicher, dass alle Änderungen zwischen den Dokumentversionen deutlich sichtbar sind und überprüft werden können.

<Warning>
**Hinweis:** Das Hochladen der neuen Version muss durch einen Klick auf **"Bestätigen"** in der unteren rechten Ecke des Fensters abgeschlossen werden. Vor der Bestätigung kann die hochgeladene Version noch gelöscht werden. Nach der Bestätigung kann die neue Version **nicht mehr gelöscht** werden, und die Zahlenprüfung wird automatisch aktualisiert und erneut überprüft.
</Warning>
