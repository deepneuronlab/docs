## Sprache der Anwendung ändern

DNL Audit unterstützt mehrere Sprachen. Die Spracheinstellung wirkt sich auf die gesamte Benutzeroberfläche aus, einschließlich der Menüs, Buttons und Systemmeldungen.

### Unterstützte Sprachen

| Sprache | Locale-Code |
|---|---|
| Deutsch | `de-DE` |
| English (Englisch) | `en-GB` |
| Français (Französisch) | `fr-FR` |
| Italiano (Italienisch) | `it-IT` |
| Español (Spanisch) | `es-ES` |
| Português (Portugiesisch) | `pt-PT` |

<Steps>
  <Step title="Menü öffnen">
    Klicken Sie auf Ihr **Profilmenü** in der oberen rechten Ecke der Anwendung.
  </Step>
  <Step title="Sprache auswählen">
    Wählen Sie die gewünschte Sprache aus der verfügbaren Liste.
  </Step>
  <Step title="Änderung übernehmen">
    Die Anwendung wird sofort in der gewählten Sprache angezeigt. Die Einstellung wird für Ihr Benutzerkonto gespeichert.
  </Step>
</Steps>

<Note>
Die Spracheinstellung wird sowohl in Ihrem Benutzerprofil als auch lokal im Browser gespeichert. Beim nächsten Login wird Ihre bevorzugte Sprache automatisch verwendet.
</Note>
