## Benutzer einem Auftrag zuweisen

Jeder Auftrag kann mehreren Benutzern zugewiesen werden. Die Zuweisung erfolgt über die Auftragseinstellungen.

<Steps>
  <Step title="Auftragseinstellungen öffnen">
    Navigieren Sie zum gewünschten Auftrag in der **Auftragsübersicht** und öffnen Sie die **Zugriffsrechte**.
  </Step>
  <Step title="Benutzer hinzufügen oder entfernen">
    Wählen Sie die Benutzer aus, die Zugriff auf den Auftrag erhalten sollen. Sie können Benutzer auch wieder entfernen.
  </Step>
</Steps>

## Berechtigungen

Die verfügbaren Berechtigungen hängen von Ihrer Rolle innerhalb der Organisation ab:

| Berechtigung | Beschreibung |
|---|---|
| **Mandate erstellen** | Neue Mandate anlegen |
| **Mandate bearbeiten** | Bestehende Mandate ändern |
| **Mandate löschen** | Mandate und zugehörige Aufträge entfernen |
| **Aufträge erstellen** | Neue Aufträge innerhalb eines Mandats anlegen |
| **Aufträge löschen** | Bestehende Aufträge entfernen |
| **Benutzer verwalten** | Benutzerzuweisungen für Aufträge ändern |
| **Alle Mandate einsehen** | Zugriff auf alle Mandate der Organisation |
| **Alle Aufträge einsehen** | Zugriff auf alle Aufträge der Organisation |

<Note>
Gruppenadministratoren können Benutzer und deren Berechtigungen über das Admin-Panel verwalten. Wenden Sie sich an Ihren Gruppenadministrator, falls Sie zusätzliche Berechtigungen benötigen.
</Note>
