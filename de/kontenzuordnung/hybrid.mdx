## Was ist ein hybrides Konto?

Ein hybrides Konto hat **unterschiedliche Taxonomiezuordnungen** für das aktuelle Jahr und das Vorjahr. Dies wird verwendet, wenn sich die Klassifizierung eines Kontos zwischen den Berichtsperioden geändert hat.

## Hybrides Konto erstellen

<Steps>
  <Step title="Kontoeditor öffnen">
    Klicken Sie auf das Drei-Punkte-Menü (⋯) auf einer Kontokarte und wählen Sie **„Bearbeiten"**, oder wählen Sie **„Hybrides Konto"** direkt.
  </Step>
  <Step title="Hybrid-Modus aktivieren">
    Schalten Sie den **„Hybrides Konto"**-Schalter im Editor-Dialog um.
  </Step>
  <Step title="Kategorien zuweisen">
    Wählen Sie eine Kategorie für die **Zuordnung aktuelles Jahr** und eine andere Kategorie für die **Zuordnung Vorjahr**.
  </Step>
  <Step title="Speichern">
    Bestätigen Sie die Zuordnung. Das Konto erscheint in beiden Taxonomiepositionen im zugeordneten Bereich.
  </Step>
</Steps>

<Note>
Die Option für hybride Konten ist nur verfügbar, wenn sowohl Werte für das aktuelle Jahr als auch für das Vorjahr vorhanden sind.
</Note>

## Visueller Indikator

Nicht zugeordnete Kontokarten zeigen ein **Hybrid-Symbol**, wenn ein Jahr bereits zugeordnet ist, das andere aber noch nicht. Ein Klick auf das Symbol navigiert zum zugeordneten Jahr im Taxonomiebaum.

## Konten erstellen und bearbeiten

Sie können auch neue Konten erstellen oder bestehende bearbeiten:

1. Klicken Sie auf **„Konto hinzufügen"** im Aktionsmenü, oder **„Bearbeiten"** bei einem bestehenden Konto
2. Geben Sie **Name**, **Nummer**, **Aktuelles Jahr** und **Vorjahr** ein
3. Aktivieren Sie optional **Hybrides Konto** und wählen Sie separate Kategoriezuordnungen
4. Klicken Sie auf **Speichern**

<Warning>
Kontonummern müssen eindeutig sein. Wenn eine Nummer bereits verwendet wird, erscheint die Fehlermeldung: „Kontonummer wird bereits verwendet."
</Warning>
